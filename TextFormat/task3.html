Text Format Task 3

This document demonstrates the use of superscript and subscript in HTML to format mathematical expressions and chemical formulas. Superscript is used to display text slightly above the normal line of text, while subscript is used to display text slightly below the normal line of text. These formatting techniques are essential for presenting scientific and mathematical information clearly and accurately.


  1. (x + y)2 = x2 + y2 + 2xy
  2. (a - b)2 = a2 - 2ab + b2
  3. a2 + b2 = (a + b)2 - 2ab
  4. (a + b + c)2 = a2 + b2 + c2 + 2ab + 2bc + 2ca
  5. (a + b)3 = a3 + b3 + 3ab2 + 3a2b
  6. 2H2 + O2 → 2H2O
  7. S8 + 12O2 → 8SO3
  8. 2Na + 2H2O → 2NaOH + H2
